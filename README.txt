Настройка и запуск:

Клонируйте репозиторий на локальную машину: git clone https://github.com/gedovirhir/ArtMax_Test
Вставьте в файле .env свои даннные (данные базы данных) и, в том числе свой телеграмм id (его можно узнать в боте https://t.me/getmyid_bot) для получения прав администратора.
Убедитесь что на вашей машине установлен docker и docker-compose
Находясь в папке проекта запустите команды в терминале: 
    docker-compose build
    docker-compose up -d
Далее зайдите в бота в тг: https://t.me/reviewAlers_bot
